## 功能特色

### 🍅 核心番茄鐘功能
1. **完整的番茄鐘週期**：25分鐘專注 + 5分鐘短休息，每4個循環後有15分鐘長休息
2. **精確計時**：秒級精確計時，支援暫停和重新開始
3. **自動切換**：專注時間和休息時間自動切換
4. **會話統計**：追蹤完成的專注時段數量

### 🎯 任務整合功能
1. **任務選擇**：可以選擇特定任務進行專注
2. **狀態同步**：番茄鐘狀態與任務狀態自動同步
3. **快速啟動**：從任務列表一鍵開始番茄鐘
4. **智慧提醒**：完成番茄鐘時自動更新任務進度

### 🎨 用戶體驗
1. **視覺化進度**：圓形進度條顯示計時進度
2. **動畫效果**：計時時的脈衝動畫增強專注感
3. **清晰狀態**：當前模式和剩餘時間一目了然
4. **便捷操作**：大按鈕設計，易於操作

### 📱 界面設計
1. **Material Design 3**：採用最新的設計規範
2. **響應式布局**：適配不同螢幕尺寸
3. **無障礙設計**：支援螢幕閱讀器和鍵盤導航
4. **深色模式**：自動適配系統主題

## 使用流程

### 🚀 完整工作流程
1. **選擇任務**：在任務頁面或番茄鐘頁面選擇要專注的任務
2. **開始專注**：點擊開始按鈕，進入25分鐘專注時間
3. **休息提醒**：專注時間完成後自動進入休息模式
4. **循環進行**：按照番茄鐘技術進行完整的工作循環
5. **任務完成**：可以在任何時候標記任務為完成

### 📋 任務管理流程
1. **從任務開始**：在任務列表點擊"開始番茄鐘"按鈕
2. **自動設置**：系統自動設置該任務為當前專注任務
3. **狀態更新**：任務狀態自動更新為"進行中"
4. **專注追蹤**：每完成一個番茄鐘，系統記錄進度

### 🔄 彈性切換
1. **任務切換**：可以隨時切換到不同任務
2. **模式切換**：可以手動切換專注/休息模式
3. **計時控制**：支援暫停、跳過、重置操作

## 技術實現

### 🏗️ 架構設計
- **狀態管理**：Riverpod 提供響應式狀態管理
- **數據持久化**：SharedPreferences 本地儲存

- **TaskSelectionDialog**：任務選擇界面
- **CircularProgressPainter**：自定義圓形進度繪製

```
任務選擇 → 設置當前任務 → 開始番茄鐘 → 完成專注時段 → 更新任務狀態
    ↓                    ↓              ↓                ↓
本地儲存 ←——————————— 狀態同步 ←——————— 自動切換 ←——————— 進度追蹤
```
