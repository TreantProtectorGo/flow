# 專注番茄 - 功能演示指南

## 🎯 應用概覽

專注番茄是一款結合任務管理和番茄鐘技術的效率應用，幫助用戶更好地管理時間和提高專注力。

## 🚀 核心功能演示

### 1. 任務管理功能

#### 添加任務
1. 打開**任務頁面**
2. 點擊右下角的 **+** 按鈕
3. 填寫任務資訊：
   - 任務標題（必填）
   - 任務描述（選填）
   - 預估番茄鐘數量（1-20個）
   - 優先級（高/中/低）
4. 點擊**新增**按鈕

#### 編輯任務
1. 點擊任務卡片
2. 修改任務資訊
3. 可以更改任務狀態（待辦/進行中/已完成）
4. 點擊**更新**按鈮

#### 任務狀態管理
- **狀態切換**：點擊任務左側的圓形按鈕
  - ⭕ 待辦事項 → ▶️ 進行中 → ✅ 已完成
- **刪除任務**：點擊任務右上角選單 → 選擇"刪除"
- **AI分析**：點擊選單 → 選擇"AI分析"查看任務建議

### 2. 番茄鐘功能

#### 基本操作
1. 切換到**番茄鐘頁面**
2. 使用控制按鈕：
   - **開始/暫停**：控制計時器
   - **重置**：重置當前計時
   - **跳過**：跳過當前階段

#### 番茄鐘週期
- **專注時間**：25分鐘專注工作
- **短休息**：5分鐘短暫休息
- **長休息**：每4個專注時段後，15分鐘長休息

#### 視覺化進度
- **圓形進度條**：顯示當前階段進度
- **脈衝動畫**：計時時的動態效果
- **時間顯示**：大字體顯示剩餘時間
- **模式指示**：清楚顯示當前階段

### 3. 任務與番茄鐘整合

#### 從任務開始番茄鐘
1. 在**任務頁面**找到要專注的任務
2. 點擊任務卡片右側的 **⏰** 按鈕
3. 系統自動：
   - 設置該任務為當前專注任務
   - 跳轉到番茄鐘頁面
   - 顯示確認訊息
4. 點擊**開始**按鈕開始專注

#### 在番茄鐘頁面選擇任務
1. 在**番茄鐘頁面**點擊當前任務區域的**切換**按鈕
2. 從可用任務列表中選擇
3. 或點擊**清除選擇**移除當前任務

#### 自動狀態同步
- 選擇任務後，任務狀態自動更新為"進行中"
- 完成番茄鐘時，系統記錄專注進度
- 任務完成時，自動清除當前選擇

## 🔧 使用技巧

### 高效工作流程
1. **規劃階段**：在任務頁面添加當日所有任務
2. **優先排序**：設置任務優先級，優先處理重要任務
3. **專注執行**：使用番茄鐘技術保持專注
4. **適時休息**：遵循番茄鐘的休息週期
5. **定期回顧**：查看統計頁面分析工作效率

### 任務管理建議
- **細化任務**：將大任務分解為小任務
- **預估時間**：合理估算所需番茄鐘數量
- **及時更新**：隨時更新任務狀態和進度
- **定期整理**：刪除已完成或不再需要的任務

### 專注技巧
- **環境準備**：確保工作環境安靜舒適
- **目標明確**：每個番茄鐘都有明確的任務目標
- **避免干擾**：專注時間內避免查看通知或社交媒體
- **尊重節奏**：嚴格遵循工作和休息的節奏
